# 🧵 Java Multithreading & Core Concepts Practice

This repository contains a collection of Java programs developed during college coursework to strengthen core programming concepts. It primarily focuses on multithreading, along with essential topics like OOP principles, exception handling, data structures, recursion, and socket programming.

The goal of this repository is to build a strong foundation in Java through hands-on practice and practical implementations.

---

## 📌 Topics Covered

### 🔹 Core Java Basics

* Ascending sorting
* Sum of integers
* Finding biggest of 3 numbers
* Prime number check
* Quadratic equation solver

### 🔹 Object-Oriented Programming (OOP)

* Constructor and method overloading
* Dynamic polymorphism
* Nested classes
* Implementing interfaces

### 🔹 Strings & Recursion

* String palindrome check
* String and number operations
* Recursive Fibonacci implementation

### 🔹 Exception Handling

* Custom exception implementation

### 🔹 Data Structures

* Stack and Queue implementation

### 🔹 Multithreading (Main Focus 🚀)

* Thread creation and execution
* Runnable interface usage
* Inter-thread communication
* Odd-even threads
* Time gap threads
* Multiple thread handling

### 🔹 Networking

* TCP/IP socket application

### 🔹 Miscellaneous

* Geometric figure calculations
* Various function implementations

---

## 📂 File Structure Overview

Each .java file in this repository represents a standalone program demonstrating a specific concept:

* ascending_sorting.java → Sorting logic
* biggest_of_3_nums.java → Conditional statements
* constructor_and_overloading.java → OOP basics
* customized_exception.java → Exception handling
* dynamic_polymorphism.java → Runtime polymorphism
* inter_thread.java, odd_even_threads.java, time_gap_threads.java → Multithreading
* runnable_interface.java → Runnable implementation
* tcp_ip_socket_application.java → Networking basics
* stack_queue.java → Data structure implementation
* recursive_fibonacci.java → Recursion
* string_palindrome.java → String handling

…and more.

## ⚙️ How to Run

1. Make sure Java is installed:
   java -version

2. Compile any file:
   javac filename.java

3. Run the program:
   java filename
